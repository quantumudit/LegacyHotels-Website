## Overview

The objective of this project is to design and develop a responsive website for a fictitious hotel chain called **Legacy Hotels** that delivers an exceptional user experience to potential guests. The website is visually appealing, easy to navigate, and designed to engage visitors with the features and amenities of Legacy Hotels.

The website design focuses on showcasing the luxurious accommodations, exceptional dining options, and top-tier customer service offered by Legacy Hotels. Additionally, it features social media integrations to encourage visitor engagement.

The website is built with responsive design principles, ensuring that it is accessible and optimized for viewing on various devices, including desktop computers, tablets, and mobile phones. The responsive design will enable potential guests to access the website and view Legacy Hotels' offerings regardless of the device they use.

## Prerequisites

A solid understanding of the following skills is recommended to fully grasp the concepts and processes involved in this project:

- HTML
- Basics of CSS, including Flexbox and Media Queries
- Responsive Design principles

Having proficiency in these skills will help to ensure that the website is developed with proper coding practices and adheres to modern design standards, including responsive design.
